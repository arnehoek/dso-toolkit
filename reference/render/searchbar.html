Example: Open/Close

Gebruik dit voorbeeld om te zien hoe de animatie in zijn werk gaat.

Zoekbalk gesloten

De zoekbalk dient vanuit zichzelf de class .dso-searchbar te hebben en kan worden uitgebreid met de modifier classes .dso-searchbar-closed of .dso-searchbar-open. Het togglen (door scripting) van deze modifier classes start een animatie de de zoekbalk opent of sluit.

Zoekbalk open

Om een zoekbalk te openen dient de class .dso-searchbar-closed doormiddel van scripting vervangen te worden door de class .dso-searchbar-open. Dit triggert een animatie die de zoekbalk opent.
